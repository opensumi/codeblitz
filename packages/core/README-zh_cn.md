# CodeBlitz

> 基于 OpenSumi 的纯前端版本 IDE 框架

## 核心功能

* 开箱即用，通过 React 组件方式只需几行代码即可运行一个无容器版本的 IDE
* 内置 BrowserFS，根据业务形式定制 FileSystem，实现文件读写
* 通过 Worker 扩展丰富 IDE 功能，支持扩展的自主安装
