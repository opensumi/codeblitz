# Quick Start

## Clone

```bash

git clone git@github.com:opensumi/codeblitz.git
cd codeblitz
```

## Install

```bash
yarn
yarn run init
```

## Start

```bash
yarn start
```

CodeBlitz 有很多集成场景，在开发的时候需要模拟这些场景，你也可以直接通过命令行启动这些场景：

- diff-viewer 行内的 diff 查看器
  `yarn diff-viewer`
- code
  `yarn code`
- provider 直接渲染编辑器组件、Diff 组件的例子
  使用 `CodeEditor`/`DiffEditor`。  
  `yarn provider`
